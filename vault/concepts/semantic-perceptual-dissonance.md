# Semantic Perceptual Dissonance

> *Auto-generated stub. Edit this file to add more details.*

A phenomenon where high-level semantic priors from LLMs become incorrectly entangled with low-level numerical time-series dynamics.


## Why It Matters

Identifies a fundamental performance bottleneck when using LLMs for time-series forecasting due to modality granularity mismatches.

## Evidence

> high-level abstract semantics provided by the LLM become inappropriately entangled with the low-level, fine-grained numerical dynamics of time series
