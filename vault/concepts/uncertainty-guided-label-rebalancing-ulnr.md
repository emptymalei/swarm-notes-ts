# Uncertainty-Guided Label Rebalancing (uLNR)

> *Auto-generated stub. Edit this file to add more details.*

A mechanism that probabilistically relabels safe-labeled data points with high predicted uncertainty as unsafe to enrich the minority class in imbalanced datasets.
