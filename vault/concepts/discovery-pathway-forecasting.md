# Discovery pathway forecasting

> *Auto-generated stub. Edit this file to add more details.*

A task-based framework for identifying and grounding the sequences of enabling contributions that lead to a specific scientific discovery.


## Why It Matters

This task frames scientific discovery as a structural dependency problem rather than simple link prediction or retrieval, shifting how scientific progress can be evaluated.

## Evidence

> In this paper, we introduce discovery pathway forecasting: given a target scientific contribution and the prior literature available at a specified time, the task is to (1) identify the enabling contributions required to realize it and (2) ground each in prior work when such prior work exists.
