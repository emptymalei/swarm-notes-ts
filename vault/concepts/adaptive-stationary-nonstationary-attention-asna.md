# Adaptive Stationary-Nonstationary Attention (ASNA)

> *Auto-generated stub. Edit this file to add more details.*

An attention mechanism that adaptively balances common and instance-specific dependencies by processing both normalized and raw sequences based on instance-level non-stationarity.


## Why It Matters

It is the central technical innovation proposed to solve the tension between modeling common patterns and instance-specific dependencies in non-stationary time series.

## Evidence

> At its core is Adaptive Stationary-Nonstationary Attention (ASNA), which captures common dependencies from normalized sequences and specific dependencies from raw sequences, and adaptively fuses them according to instance-level non-stationarity.
