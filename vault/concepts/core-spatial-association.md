# Core Spatial Association

> *Auto-generated stub. Edit this file to add more details.*

A framework that uses random matrix theory to isolate genuine spatial dependencies by filtering out dominant temporal signals in spatial time series.


## Why It Matters

This provides a principled method for disentangling spatial dependence from temporal co-evolution in spatiotemporal time series, which is a common bottleneck in climate and geophysical modeling.

## Evidence

> We introduce a Random Matrix Theory (RMT)-based framework to isolate \"core spatial association\" by suitably trimming out strong but routine temporal signals while preserving spatial signals.
