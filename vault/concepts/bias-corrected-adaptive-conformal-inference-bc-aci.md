# Bias-Corrected Adaptive Conformal Inference (BC-ACI)

> *Auto-generated stub. Edit this file to add more details.*

A conformal inference framework that integrates online bias estimation to dynamically re-center prediction intervals under regime shifts.


## Why It Matters

It introduces a novel framework that corrects forecast bias in online settings to improve prediction interval quality beyond symmetric expansion, which is a common limitation in standard conformal inference for time series.

## Evidence

> BC-ACI corrects nonconformity scores before quantile computation and re-centers prediction intervals, addressing the root cause of miscalibration rather than its symptom.
