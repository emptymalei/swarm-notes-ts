# Slope Attenuation Bias (Ensemble Forecasting)

> *Auto-generated stub. Edit this file to add more details.*

A systematic bias in conditional reliability diagnostics for ensemble forecasts caused by finite sampling noise, resulting in the attenuation of observed-vs-predicted slopes.


## Why It Matters

It provides a fundamental, quantifiable correction to standard reliability diagnostics that is often misinterpreted as model deficiency in ensemble forecasting.

## Evidence

> We present a unified framework for slope attenuation caused by finite-ensemble sampling noise, which affects conditional diagnostics for ensemble means, spreads, and probabilities.
