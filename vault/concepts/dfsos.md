# Deflation-Free Sparse Optimal Scoring (DFSOS)

> *Auto-generated stub. Edit this file to add more details.*

A simultaneous estimation framework for sparse linear discriminant analysis using global orthogonality constraints to avoid deflation-based error propagation.


## Why It Matters

DFSOS addresses the error propagation issue inherent in sequential deflation-based strategies for discriminant analysis.

## Evidence

> We propose a novel approach that estimates all discriminant vectors simultaneously under an explicit global orthogonality constraint, which we call Deflation-Free Sparse Optimal Scoring (DFSOS).
