# Weight-Space World Model

> *Auto-generated stub. Edit this file to add more details.*

A framework for world modeling that defines system states as weights of an implicit neural representation rather than standard vector-based latents.


## Why It Matters

It shifts the world model paradigm from opaque vector-based latents to weight-space coordinate representations, enabling structural disentanglement and decoder-free inference.

## Evidence

> NOVA, a world modelling framework that represents the system state as the weights and biases of an auxiliary coordinate-based implicit neural representation (INR).
