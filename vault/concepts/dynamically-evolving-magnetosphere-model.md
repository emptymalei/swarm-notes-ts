# Dynamically Evolving Magnetosphere Model

> *Auto-generated stub. Edit this file to add more details.*

An extension of the rotating vector model where stochastic perturbations to the effective magnetic axis account for PA variations in repeating FRBs.


## Why It Matters

This model explains observed PA distributions and the lack of periodicity in repeating FRBs through stochastic magnetic axis variations.

## Evidence

> We interpret these properties by extending the rotating vector model to include a dynamically evolving magnetosphere, in which the effective magnetic axis varies from burst to burst due to stochastic perturbations.
