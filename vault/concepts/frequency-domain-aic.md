# Frequency-Domain AIC

> *Auto-generated stub. Edit this file to add more details.*

A frequency-domain information criterion used for the automatic selection of smoothing bandwidths in nonparametric spectral density estimation.


## Why It Matters

It proposes an objective, automatic method for nonparametric spectral density bandwidth selection as an alternative to subjective visual smoothing.

## Evidence

> presenting evidence that using what is arguably the simplest and most straightforward frequency-domain version of the AIC for the automatic determination of an appropriate bandwidth enables results that are comparable to those obtained using the standard parametric approach.
