# Nonparametric Hawkes Process for Extreme Values

> *Auto-generated stub. Edit this file to add more details.*

A Bayesian nonparametric approach using a Dirichlet process to learn temporal excitation patterns within a Hawkes point process for extreme events.


## Why It Matters

This integration enables flexible learning of clustering dynamics in nonstationary extreme event processes without assuming rigid parametric forms, which is a major hurdle in extreme value theory.
