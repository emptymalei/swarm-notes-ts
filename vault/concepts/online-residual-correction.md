# Online Residual Correction

> *Auto-generated stub. Edit this file to add more details.*

A deployment strategy for time-series forecasting where a pre-trained model's predictions are adjusted online using the difference between the actual observed values and the model's previous predictions.


## Why It Matters

It is a specific technique used to adapt frozen models in deployment, potentially reducing computational overhead while maintaining accuracy, which is a key practical consideration.
