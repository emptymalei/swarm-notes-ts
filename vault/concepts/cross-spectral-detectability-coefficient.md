# Cross-spectral detectability coefficient

> *Auto-generated stub. Edit this file to add more details.*

A quartic-order cross-contribution metric derived from the spectral matrix that enables detection of non-equilibrium behavior in linear systems where single-channel measures fail.


## Why It Matters

It overcomes the 'single-channel impossibility' for detecting time-irreversibility in linear Gaussian systems by leveraging off-diagonal spectral information.

## Evidence

> the cross-spectral detectability coefficient Scross (the leading quartic-order cross contribution) is exactly independent of the observed timescales
