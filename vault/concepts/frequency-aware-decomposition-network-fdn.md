# Frequency-aware Decomposition Network (FDN)

> *Auto-generated stub. Edit this file to add more details.*

A forecasting architecture that spectrally decomposes signals to model low-frequency components deterministically and high-frequency residuals as conditional distributions.


## Why It Matters

The architecture introduces a distinct hybrid approach to forecasting by separating signals into deterministic and probabilistic components based on frequency.
