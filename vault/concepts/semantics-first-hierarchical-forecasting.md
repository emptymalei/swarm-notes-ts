# Semantics-First Hierarchical Forecasting

> *Auto-generated stub. Edit this file to add more details.*

A hierarchical framework that predicts future scene representations in a frozen vision foundation model's feature space before synthesizing RGB frames.


## Why It Matters

It introduces a modular paradigm for video prediction, decoupling semantic dynamic modeling in a frozen feature space from pixel-level synthesis, which is a major shift from end-to-end approaches.

## Evidence
