# Adaptive Long-Run Variance Thresholding

> *Auto-generated stub. Edit this file to add more details.*

A thresholding procedure for sparse covariance estimation that adjusts entry-specific thresholds using estimated long-run variance to account for temporal autocorrelation.


## Why It Matters

It addresses the failure of standard thresholding methods when applied to high-dimensional time series with temporal dependence by replacing naive i.i.d. assumptions.
