# LTE-ODE

> *Auto-generated stub. Edit this file to add more details.*

A Neural ODE framework that treats local truncation error as an unsupervised inductive bias to adaptively switch between continuous evolution and discrete shock compensation.


## Why It Matters

It resolves the continuity-shock dilemma in Neural ODEs by repurposing numerical integration errors as an inductive bias to distinguish between stable evolution and abrupt anomalies.

## Evidence
