# Temporal Operator Attention (TOA)

> *Auto-generated stub. Edit this file to add more details.*

A time-series attention variant that replaces convex combinations with learnable sequence-space operators for improved oscillatory signal representation.


## Why It Matters

TOA overcomes the simplex-constrained mixing bottleneck inherent in softmax-based attention mechanisms by allowing signed mixing.
