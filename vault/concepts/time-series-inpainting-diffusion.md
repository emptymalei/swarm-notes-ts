# Time-series Inpainting Diffusion

> *Auto-generated stub. Edit this file to add more details.*

A conditional diffusion framework that treats future time-series values as missing regions to be reconstructed via iterative denoising.


## Why It Matters

It introduces a paradigm shift in time-series forecasting by reinterpreting temporal extrapolation as a generative image inpainting task.
