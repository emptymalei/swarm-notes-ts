# Multimodal Transformer Based Generic Mixture Density Network

> *Auto-generated stub. Edit this file to add more details.*

A deep learning architecture that fuses dynamic spectra and timeseries data using parallel transformer encoders and a generic mixture-density output layer to estimate scattering timescales.


## Why It Matters

It provides a specialized architecture for integrating time-series and spectral data with a density-based output layer designed to handle zero-inflated phenomena, which is a common challenge in astrophysical parameter estimation.

## Evidence
