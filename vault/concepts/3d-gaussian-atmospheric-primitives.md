# 3D Gaussian Atmospheric Primitives

> *Auto-generated stub. Edit this file to add more details.*

A spatial representation technique that models atmospheric grid points as learnable 3D Gaussians to facilitate continuous resolution adaptation.


## Why It Matters

Replaces rigid grid representations with continuous, parameterizable Gaussian primitives, enabling arbitrary-resolution forecasting which is a fundamental shift in atmospheric modeling.

## Evidence

> latitude-longitude grid points are treated as centers of 3D Gaussians. A generative 3D Gaussian prediction scheme is introduced to estimate key parameters, including covariance, attributes, and opacity, for unseen samples
