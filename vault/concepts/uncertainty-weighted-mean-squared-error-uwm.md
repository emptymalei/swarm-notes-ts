# Uncertainty Weighted Mean Squared Error (UWM)

> *Auto-generated stub. Edit this file to add more details.*

A loss function for event-aware forecasting that integrates block interval regression into TPP objectives by weighting uncertainty with homoscedasticity.


## Why It Matters

The loss function specifically addresses the dual challenge of event timing and classification in event-driven financial systems, bridging the gap between point process modeling and regression-based interval prediction.

## Evidence
