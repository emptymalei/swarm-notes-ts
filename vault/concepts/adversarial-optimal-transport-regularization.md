# Adversarial Optimal Transport Regularization

> *Auto-generated stub. Edit this file to add more details.*

A training framework for neural emulators that employs optimal transport objectives to align the statistical distributions and attractors of generated and reference chaotic trajectories.

This is a novel, high-level methodological paradigm for training emulators on chaotic systems that bypasses the limitations of point-wise squared-error losses.
