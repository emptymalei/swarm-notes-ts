# Cross-sensor RGB spectrogram

> *Auto-generated stub. Edit this file to add more details.*

A visual anomaly detection method mapping three concurrent magnetometer power spectra into a single RGB image to isolate unique spectral energy from coherent signals.


## Why It Matters

It provides a novel, sensor-agnostic visual diagnostic tool for multi-sensor triads by mapping inter-sensor coherence to grayscale and unique spectral noise to color.

## Evidence

> The standard analysis pipeline reduces each sensor to an independent power spectrum, discarding any inter-sensor structure that is itself diagnostic of measurement health and of localised magnetic activity.
