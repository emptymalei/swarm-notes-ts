# Wasserstein Distributionally Robust Estimation

> *Auto-generated stub. Edit this file to add more details.*

A signal estimation framework that minimizes the worst-case Conditional Value-at-Risk (CVaR) over a type-2 Wasserstein ambiguity set of distributions.


## Why It Matters

Provides a formal framework for risk-sensitive signal estimation that explicitly addresses distribution shift via robust optimization, a recurring challenge in time-series forecasting.

## Evidence

> We propose a distributionally robust approach to risk-sensitive estimation of an unknown signal x from an observed signal y ... assumed to belong to a given type-2 Wasserstein ball of distributions.
