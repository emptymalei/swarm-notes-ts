# Imprecise Poisson Process

> *Auto-generated stub. Edit this file to add more details.*

A game-theoretic point process model that replaces exact rate assumptions with rate intervals, enabling more robust forecasting under incomplete information.


## Why It Matters

It provides a game-theoretic extension to traditional point process modeling by handling rate uncertainty via interval bounds.

## Evidence

> I replace this assumption with the less stringent -- and arguably more realistic -- one that the available information about the process takes the form of bounds on the rate rather than a single, exact value. The resulting process has properties similar to the standard, 'precise' Poisson process, albeit with an imprecise flavour to them, thus justifying the moniker 'imprecise Poisson process'.
