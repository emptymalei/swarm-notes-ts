# Time Rescaling Theorem for Event Forecasting

> *Auto-generated stub. Edit this file to add more details.*

A method for modeling asynchronous event streams by learning a bijection between irregular event sequences and unit-rate exponential noise.


## Why It Matters

It transforms the problem of modeling irregularly sampled events into a standardized mapping to uniform noise, providing a principled foundation for multivariate event stream modeling that avoids expensive numerical quadrature.
