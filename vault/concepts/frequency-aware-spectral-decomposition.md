# Frequency-Aware Spectral Decomposition

> *Auto-generated stub. Edit this file to add more details.*

A forecasting paradigm that partitions signal components into frequency-specific bands, using deterministic heads for trend and probabilistic heads for high-frequency residuals.


## Why It Matters

The dual-head architecture separating low-frequency deterministic components from high-frequency probabilistic residuals provides a robust paradigm for modeling multi-modal, vibration-heavy time series.

## Evidence

> we propose a Frequency-aware Decomposition Network (FDN) for short-term forecasting of vibration-rich wrench from proprioceptive history. FDN predicts spectrally decomposed wrench with asymmetric deterministic and probabilistic heads
