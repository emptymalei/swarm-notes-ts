# Posterior-Weighted Gaussian Process

> *Auto-generated stub. Edit this file to add more details.*

A Gaussian process variant using a modulated posterior kernel to model approximately periodic time series with inter-repetition variability.


## Why It Matters

Provides a principled mechanism for decoupling stable intra-repetition structural features from inter-repetition variability in quasi-periodic time series.

## Evidence

> The model is based on a GP whose posterior is modulated by a novel kernel.
