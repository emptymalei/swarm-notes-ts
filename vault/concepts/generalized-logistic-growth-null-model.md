# Generalized Logistic Growth Null Model

> *Auto-generated stub. Edit this file to add more details.*

A statistical null model for stationary time-series growth-rate distributions that accounts for heavy-tailed or tent-shaped deviations from normality.


## Why It Matters

Provides a robust, non-normal baseline for growth-rate distributions that challenges standard Gaussian assumptions in stationary time-series analysis.

## Evidence

> Systems characterized by stationary Gamma or heavy-tailed abundance distributions produce log-growth-rate distributions well described by a generalized logistic distribution, which can describe tent-shaped or nearly normal datasets and serves as a useful null model for these observables.
