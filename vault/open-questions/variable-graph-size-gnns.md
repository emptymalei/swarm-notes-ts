title: Supporting Variable Graph Sizes
---

**Background:** Network traffic is inherently dynamic, often resulting in topologies where the number of entities and connections fluctuates over time.

**Question / Future Work:** Addressing the limitation of static graph structures in current Graph Neural Network (GNN) architectures to allow for variable graph sizes in network traffic forecasting.

**Why It Matters:** Static graph constraints significantly limit the real-world deployment of GNN-based forecasting models for network traffic.

**Evidence:** Our future work will focus on ... supporting variable graph sizes, which is necessary to better model the non-static nature of network traffic.
