**Background:** Hybrid AC-HVDC grid operations often face operational limits under contingency conditions. Ensuring system stability during these events remains a critical requirement for multi-terminal grid control architectures.

**Question / Future Work:** The current stochastic optimal power flow framework focuses on wind power uncertainty and requires extension to incorporate N-1 security contingencies into the chance-constrained optimization and subsequent adaptive droop gain extraction.

**Why It Matters:** Reliability and grid security in the face of equipment failure are essential for the practical deployment of multi-terminal HVDC systems.

**Evidence:** Future work will focus on extending this framework to incorporate N-1 security contingencies
