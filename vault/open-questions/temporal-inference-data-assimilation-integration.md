title: Temporal Inference-Data Assimilation Integration
---

**Background:** Latent-space temporal models are often trained on simulation-derived trajectories and deployed on true observational data, leading to a simulation-to-reality gap that degrades predictive accuracy.

**Question / Future Work:** There is an explicit need to integrate temporal inference frameworks with multi-scale data assimilation to reconcile discrepancies between simulated priors and true system dynamics during online extrapolation.

**Why It Matters:** This addresses the fundamental simulation-to-reality gap in physics-informed machine learning, enabling model adaptation to real-world sensor feedback.
