title: Machine Learning for Microstructure Vetting
---

**Background:** Current search pipelines for pulsar microstructures require manual inspection of diagnostic plots to confirm candidate validity, creating a processing bottleneck. Machine learning approaches are increasingly being explored to automate this classification task.

**Question / Future Work:** There is a need to develop robust machine learning-based classification techniques to automatically vet candidate quasi-periodic microstructures generated by search pipelines. Applying models to diagnostic outputs would eliminate human bottlenecks and enable scalable analysis of large archival pulsar datasets.

**Why It Matters:** Automation is critical for efficiently processing the rapidly growing volume of pulsar survey data and identifying rare or subtle emission features.

**Evidence:** machine learning based automation of the decision making will further automate the overall search procedure and QMIST.
