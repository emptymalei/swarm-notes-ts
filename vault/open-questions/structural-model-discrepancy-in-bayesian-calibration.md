title: Structural Model Discrepancy Calibration
---

**Background:** Earth system models (ESMs) utilize parameterizations with empirical parameters to represent subgrid-scale processes, yet structural model-observation discrepancies often complicate the Bayesian inversion process used to calibrate these parameters.

**Question / Future Work:** There is a need to develop frameworks that explicitly account for structural model-form error within the Bayesian inverse problem for parameter calibration, rather than assuming no model-data discrepancy. This would enable the transition from using synthetic observation-based calibration to more complex, real-world data-informed parameterization learning.

**Why It Matters:** Ignoring structural model-form error leads to biased parameter estimates and failure to generalize when calibrated parameters are applied to real-world scenarios.
