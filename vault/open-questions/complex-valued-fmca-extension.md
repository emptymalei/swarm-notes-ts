title: Complex-Valued FMCA Extension
---

**Background:** The functional maximal correlation algorithm (FMCA) is used for constructing projection spaces based on the cross density ratio (CDR) of random processes. While FMCA has been applied to real-valued time-series, its efficacy in complex-valued domains remains to be explored.

**Question / Future Work:** There is a need to extend the FMCA-based classification framework to the complex-valued domain, which is critical for signal processing applications where phase information is essential.

**Why It Matters:** Extending to the complex domain is critical for many signal processing tasks (e.g., radar, communication signals) where phase information is essential and currently neglected by purely real-valued methods.

**Evidence:** As future work, we plan to extend the framework to the complex-valued domain for improved spectral information processing.
