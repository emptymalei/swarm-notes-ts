title: Optimizing Ensemble Anomaly Detection for Spacecraft Edge Devices
---

**Background:** Spacecraft telemetry anomaly detection is increasingly performed using on-board deep learning models to minimize communication latency, but these models must operate under severe hardware constraints. Multi-objective optimization frameworks are used to find Pareto-optimal configurations that balance detection accuracy against memory and computational overhead.

**Question / Future Work:** Investigating hybrid or ensemble methods that combine the complementary strengths of multiple detection approaches for spacecraft telemetry could improve overall detection robustness without significantly increasing resource requirements. Additionally, extending multi-objective optimization frameworks to include mission-critical objectives such as energy consumption and inference latency is necessary for robust autonomous operations.

**Why It Matters:** Spacecraft anomaly detection is highly sensitive to the trade-offs between precision, recall, and hardware resource usage; integrating additional mission-critical objectives into the optimization loop is essential for reliable autonomous operations.
