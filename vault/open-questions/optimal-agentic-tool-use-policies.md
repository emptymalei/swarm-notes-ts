title: Optimal Agentic Tool-Use Policies
---

**Background:** Autonomous agents frequently employ tools like web search to improve forecasting, but these tools can introduce noisy or contradictory information that degrades performance depending on the temporal context.

**Question / Future Work:** Research is needed to develop category-aware tool-use policies that determine when an agent should utilize search versus relying on internal parametric knowledge, optimizing for calibration and scoring metrics while managing retrieval overhead.

**Why It Matters:** This addresses the critical challenge of ensuring tool-use reliably improves agentic reasoning rather than introducing noise, which is essential for high-stakes decision-making environments.
