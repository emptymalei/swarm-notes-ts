title: Hybrid CNN-LMU architectures
---

**Background:** State-space models using Legendre polynomials provide a theoretically grounded approach to memory in recurrent neural networks. Investigating their potential in hybrid architectures remains an area of interest for improving long-term temporal dependencies.

**Question / Future Work:** Future research should investigate the integration of Legendre Memory Units (LMUs) with convolutional layers to determine if this hybrid architecture can enhance trajectory forecasting accuracy by combining spatial context with the principled memory retention properties of Legendre polynomials.

**Why It Matters:** The paper highlights this configuration as a potential avenue to overcome current limitations in integrating spatial and long-term temporal context in trajectory prediction.

**Evidence:** This suggests potential in combining CNN and LMU using Legendre polynomials.
