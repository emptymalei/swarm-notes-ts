title: Harmonization of Pollutant Units
---

**Background:** Global air-quality datasets often contain heterogeneous, provider-native monitoring units that vary by region and pollutant type. Current evaluation protocols frequently rely on simplified, normalized metrics or ignore these discrepancies, complicating direct, physically interpretable comparisons across different monitoring networks.

**Question / Future Work:** Developing standardized, robust methods to handle unit conversion and cross-pollutant harmonization is essential to move from regional, isolated monitoring to a globally coherent and physically meaningful forecasting framework. This requires establishing explicit conversion protocols that can account for the diversity in sensor hardware and provider-specific reporting formats.

**Why It Matters:** This issue is a fundamental bottleneck to achieving physically valid evaluation and cross-region model generalization in global environmental monitoring.

**Evidence:** A fully harmonized release with station-level unit audits and explicit conversion rules is an important future extension.
