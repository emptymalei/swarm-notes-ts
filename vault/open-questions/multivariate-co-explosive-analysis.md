title: Multivariate Path-Co-Explosive Analysis
---

**Background:** The current co-explosive analysis is limited to pairwise comparisons, hindering the ability to analyze systemic relationships within groups. A multivariate generalization would allow for the identification of common explosive components in larger sets of data.

**Question / Future Work:** A multivariate extension of the co-explosive analysis is required to characterize the number of common path-explosive components within a system of series, analogous to cointegration rank testing. This is essential for analyzing complex panels like metropolitan housing markets or competing tourism destinations.

**Why It Matters:** Multivariate analysis is essential for understanding systemic risk and spillover effects in interconnected economic networks which cannot be captured by pairwise analysis alone.

**Evidence:** The current co-explosive analysis is pairwise. A multivariate extension analogous to cointegration rank—characterising the number of common path-explosive components in a system of series—would be valuable.
