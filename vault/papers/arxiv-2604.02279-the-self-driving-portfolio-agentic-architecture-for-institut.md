# The Self Driving Portfolio: Agentic Architecture for Institutional Asset Management

**Authors**: Andrew Ang, Nazym Azimbayev, Andrey Kim
**Date**: 2026-04-02
**Paper ID**: [arxiv:2604.02279](https://arxiv.org/abs/2604.02279)

## Summary

This paper introduces an agentic architecture for institutional asset management where autonomous agents handle capital market assumptions, portfolio construction, and cross-evaluation. The system features a self-improving loop driven by a meta-agent that refines code and prompts based on forecast performance. Oversight is maintained through an Investment Policy Statement (IPS), which serves as a natural language governance layer to constrain autonomous investment activities.

## Key Contributions

- Introduces an agentic strategic asset allocation pipeline consisting of ~50 specialized agents that perform market assumption generation, portfolio construction, and peer critique.
- Implements a researcher agent that autonomously proposes new portfolio construction methodologies and a meta-agent that performs self-corrective code and prompt optimization based on realized returns.
- Develops a governance framework where an Investment Policy Statement (IPS) acts as the high-level constraint mechanism to steer autonomous agent behavior in institutional asset management.
