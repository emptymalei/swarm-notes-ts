# mtslearn: Machine Learning in Python for Medical Time Series

**Authors**: Zhongheng Jiang, Yuechao Zhao, Donglin Xie, Chenxi Sun, Rongchen Lu, Silu Luo, Zisheng Liang, Shenda Hong
**Date**: 2026-03-31
**Paper ID**: [arxiv:2603.29432](https://arxiv.org/abs/2603.29432)

## Summary

mtslearn is an end-to-end integrated Python toolkit designed to streamline the machine learning workflow for complex, heterogeneous medical time-series data. The framework provides a unified interface for automated data parsing and alignment, significantly reducing the manual effort required for data preparation. By offering a modular pipeline that spans from feature engineering to result visualization, it enables clinicians with limited programming expertise to implement and evaluate advanced AI models in clinical practice.

## Key Contributions

- Introduces mtslearn, an end-to-end Python toolkit designed to standardize workflows for medical time-series data analysis.
- Implements a unified data interface that automates the parsing and alignment of heterogeneous data formats (wide, long, and flat), reducing pre-processing overhead.
- Provides a modular, user-friendly pipeline integrating data reading, feature engineering, model training, and visualization to lower barriers for clinical practitioners.
