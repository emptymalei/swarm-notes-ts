# BEHAVE: A Hybrid AI Framework for Real-Time Modeling of Collective Human Dynamics

**Authors**: Helene Malyutina
**Date**: 2026-05-12
**Paper ID**: [arxiv:2605.12730](https://arxiv.org/abs/2605.12730)

## Summary

BEHAVE is a formal framework designed to model collective human dynamics by treating interacting humans as a complex dynamical system rather than independent agents. By transforming kinematic micro-signals into a directed interaction graph and subsequently into continuous behavioral fields, the system identifies phase transitions and critical states within group interactions. This hybrid approach leverages both mathematical formalization of system dynamics and neural-based perception for real-time forecasting, addressing limitations in current individual-centric AI systems.

## Key Contributions

- Introduces the BEHAVE framework, which models collective human behavior as a continuous dynamical system using physical kinematic micro-signals.
- Proposes a mathematical formulation for behavioral fields, including a tension field and criticality index, to capture emergence and phase transitions in group dynamics.
- Demonstrates a real-time forecasting pipeline for complex collective human dynamics using a neural implementation of the interaction graph-based behavioral field basis.
