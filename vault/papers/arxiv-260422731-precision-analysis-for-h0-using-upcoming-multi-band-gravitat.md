# Precision Analysis for H0 Using Upcoming Multi-band Gravitational Wave Observations

**Authors**: Md Riajul Haque Setabuddin, Ratna Koley, Supratik Pal
**Date**: 2026-04-24
**Paper ID**: [arxiv:2604.22731](https://arxiv.org/abs/2604.22731)

## Summary

This paper explores the potential of multi-band gravitational wave (GW) observations to constrain the Hubble parameter (H0) using primordial black holes as sources. By combining data from the Square Kilometre Array (SKA) and the Einstein Telescope (ET), the authors employ Fisher matrix analysis to forecast uncertainties in PBH mass and abundance, propagating these into H0 estimates. The findings indicate that such multi-band approaches offer a significant, independent pathway for precise H0 measurements, independent of traditional cosmic distance ladders.

## Key Contributions

- Proposed a multi-band gravitational wave analysis framework integrating scalar-induced and merger-induced waves from primordial black holes (PBHs).
- Forecasted constraints on the Hubble parameter (H0) using combined observations from the Square Kilometre Array (SKA) and Einstein Telescope (ET).
- Demonstrated that multi-band GW observations can achieve H0 uncertainty estimates down to O(0.1) km/s/Mpc, offering a cosmic distance ladder-independent measurement technique.
