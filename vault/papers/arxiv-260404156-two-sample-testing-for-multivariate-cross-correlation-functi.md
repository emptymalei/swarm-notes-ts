# Two-Sample Testing for Multivariate Cross-Correlation Functions with Applications to Gut-Brain Reward Learning

**Authors**: Bhaskar Ray, Tùng Bùi, William Matthew Howe, Srijan Sengupta
**Date**: 2026-04-05
**Paper ID**: [arxiv:2604.04156](https://arxiv.org/abs/2604.04156)

## Summary

This paper presents a rigorous functional data analysis (FDA) framework for performing two-sample hypothesis testing on multivariate cross-correlation functions (CCFs), shifting their usage from descriptive to inferential. The authors propose two global test statistics, an integrated version (F_int) for broad lag-domain differences and a maximum-type version (F_max) for localized variations, both derived from pointwise Hotelling T^2 statistics. By applying this method to gut-brain reward learning data, the study successfully quantifies and compares complex dynamic dependence structures between experimental groups.

## Key Contributions

- Develops a functional data analysis (FDA) framework for two-sample hypothesis testing on multivariate cross-correlation functions (CCFs).
- Constructs global test statistics (integrated F_int and maximum-type F_max) based on pointwise Hotelling T^2 statistics to assess lead-lag relationship differences.
- Demonstrates the framework's efficacy in identifying significant dopamine-locomotion coupling variations across biological sex and brain regions using gut-brain interaction datasets.
