# An astrometric search for planets in debris disk systems

**Authors**: Elisabeth M. Penderghast, Benjamin C. Bromley, Scott J. Kenyon, Joan R. Najita
**Date**: 2026-04-08
**Paper ID**: [arxiv:2604.07068](https://arxiv.org/abs/2604.07068)

## Summary

This paper investigates the potential for detecting planetary companions in 176 debris disk systems using astrometric data from the Gaia Data Release 3 (DR3) archive. By training a machine-learning model on known exoplanetary host stars with similar physical characteristics, the authors quantify the efficacy of the Gaia ruwe parameter in identifying massive companions. The study identifies promising candidate systems likely to host undiscovered planets, offering a prioritized target list for future Gaia DR4 time-series analysis.

## Key Contributions

- Evaluated the sensitivity of Gaia Data Release 3 (DR3) astrometric parameters, specifically the ruwe metric, for detecting planetary companions in 176 debris disk systems.
- Developed a machine-learning-based metric trained on known exoplanetary host stars to identify high-probability candidates for undiscovered planets within debris disk systems.
- Provided a prioritized list of debris disk host stars as primary targets for upcoming Gaia Data Release 4 time-series analyses.
