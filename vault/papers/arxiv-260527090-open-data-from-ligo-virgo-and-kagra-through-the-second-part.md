# Open Data from LIGO, Virgo, and KAGRA through the Second Part of the Fourth Observing Run

**Authors**: The LIGO Scientific collaboration, the Virgo collaboration, the KAGRA collaboration
**Date**: 2026-05-26
**Paper ID**: [arxiv:2605.27090](https://arxiv.org/abs/2605.27090)

## Summary

This paper documents the release of gravitational-wave strain data collected by the LIGO, Virgo, and KAGRA detector network during the second part of the fourth observing run (O4b). The release includes high-fidelity strain time series, auxiliary noise channels, and associated analysis products hosted on the Gravitational Wave Open Science Center (GWOSC). This dataset facilitates community-driven research and analysis using the latest version 5.0 of the Gravitational-Wave Transient Catalog.

## Key Contributions

- Provides public access to calibrated gravitational-wave strain time series from the O4b observing run (April 2024 – January 2025).
- Includes supplementary detector characterization and noise subtraction channels to support transient analysis.
- Updates GWOSC data release infrastructure to align with version 5.0 of the Gravitational-Wave Transient Catalog.
