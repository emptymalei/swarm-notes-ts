# Convolutionally Low-Rank Models with Modified Quantile Regression for Interval Time Series Forecasting

**Authors**: Miaoxuan Zhu, Yi Yu, Yuyang Li, Wei Li, Guangcan Liu
**Date**: 2026-04-17
**Paper ID**: [arxiv:2604.15791](https://arxiv.org/abs/2604.15791)

## Summary

This paper introduces LbCNNM-MQR, an interval time series forecasting framework designed to provide reliable uncertainty estimation. By augmenting the existing Learning-Based Convolution Nuclear Norm Minimization (LbCNNM) method with a modified quantile regression approach, the authors enable direct generation of prediction intervals alongside point forecasts. Furthermore, the inclusion of interval calibration techniques significantly improves the reliability and coverage of these predictions. The approach is validated across an extensive dataset of 100,000 time series, showing competitive results in uncertainty quantification.

## Key Contributions

- Proposed LbCNNM-MQR, an interval forecasting extension of the point-forecasting LbCNNM framework, by integrating modified quantile regression.
- Developed interval calibration techniques to refine the accuracy of prediction intervals generated by the model.
- Demonstrated superior predictive performance on a large-scale evaluation set containing over 100,000 real-world time series.
