# Bridging Deep Learning and Integer Linear Programming: A Predictive-to-Prescriptive Framework for Supply Chain Analytics

**Authors**: Khai Banh Nghiep, Duc Nguyen Minh, Lan Hoang Thi
**Date**: 2026-04-02
**Paper ID**: [arxiv:2604.01775](https://arxiv.org/abs/2604.01775)

## Summary

This paper presents an integrated predictive-to-prescriptive framework for supply chain optimization. The methodology utilizes advanced deep learning architectures, specifically N-BEATS and N-HiTS, to generate precise 4-week demand forecasts from granular retail transaction data. These forecasts serve as inputs for an integer linear programming model designed to minimize delivery times under strict operational, capacity, and budgetary constraints. Experimental results validate that combining high-accuracy neural forecasting with deterministic optimization yields feasible and cost-optimal logistical planning.

## Key Contributions

- Introduces a three-stage framework bridging deep learning forecasting with integer linear programming for supply chain logistics.
- Evaluates and compares N-BEATS and N-HiTS for retail demand forecasting, demonstrating superior performance over statistical benchmarks.
- Applies N-BEATS-based forecasts to a constrained integer linear program to generate cost-optimal shipping plans with budget and service guarantees.
