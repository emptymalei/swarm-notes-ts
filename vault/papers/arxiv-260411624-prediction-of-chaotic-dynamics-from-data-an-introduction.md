# Prediction of chaotic dynamics from data: An introduction

**Authors**: Luca Magri, Andrea Nóvoa, Elise Özalp, Luca Magri, Andrea Nóvoa, Elise Özalp
**Date**: 2026-04-13
**Paper ID**: [arxiv:2604.11624](https://arxiv.org/abs/2604.11624)

## Summary

This paper provides a principled, pedagogical introduction to predicting chaotic dynamics from data by merging dynamical systems theory with machine learning models. It specifically examines the performance of Echo State Networks (ESNs) and Long-Short-Term Memory (LSTM) networks in capturing the complex, long-term behavior of chaotic systems such as the Lorenz attractor. The work emphasizes a dynamical systems-oriented perspective to interpret how neural models learn and forecast non-linear time-series, supported by practical coding tutorials.

## Key Contributions

- Provides a pedagogical framework for predicting chaotic dynamical systems using machine learning, bridging classical dynamical systems theory with modern neural architectures.
- Demonstrates the application of Echo State Networks (ESN) and LSTM networks to iconic chaotic systems like the Lorenz attractor.
- Integrates dynamical systems perspective into time-series forecasting, offering tutorials for practical implementation of these methods.
