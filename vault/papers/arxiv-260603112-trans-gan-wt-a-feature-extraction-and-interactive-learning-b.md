# Trans GAN-WT: A Feature Extraction and Interactive Learning-Based Anomaly Detection Model for Wind Turbine Time Series Data

**Authors**: Jingzhe Kang
**Date**: 2026-06-02
**Paper ID**: [arxiv:2606.03112](https://arxiv.org/abs/2606.03112)

## Summary

TransGAN-WT is an anomaly detection model designed for wind turbine monitoring that integrates Transformer-based feature extraction with generative adversarial networks. The model uses autoregressive inference to capture multimodal features and a dedicated temporal extraction module to enhance learning across different time scales while minimizing redundancy. By amplifying reconstruction errors, it improves the detection of subtle anomalies, achieving statistically significant performance improvements in both F1 score and false positive rates over current state-of-the-art approaches.

## Key Contributions

- Introduces TransGAN-WT, a hybrid Transformer and GAN model for wind turbine anomaly detection that leverages autoregressive multimodal feature extraction.
- Implements a temporal feature extraction module to facilitate multi-scale interaction and reduce time redundancy.
- Achieves a 96.10% average F1 score on wind turbine datasets, outperforming state-of-the-art baselines by up to 5.84% while maintaining a low false positive rate of 0.06%.
