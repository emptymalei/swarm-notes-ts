# Private Federated Learning for High-dimensional Time Series

**Authors**: Kejun Chen, Qianqian Zhu
**Date**: 2026-04-08
**Paper ID**: [arxiv:2604.07135](https://arxiv.org/abs/2604.07135)

## Summary

This paper presents a privacy-preserving federated learning framework designed for high-dimensional vector autoregressive (VAR) models in decentralized settings. The method decomposes client dynamics into a shared low-rank structure and sparse, client-specific deviations, employing a two-stage procedure with differential privacy. Through theoretical analysis of non-asymptotic error bounds and empirical applications in macroeconomic and electricity-economy forecasting, the framework demonstrates improved predictive performance when local data is limited compared to single-client models.

## Key Contributions

- Proposes a federated learning framework for high-dimensional vector autoregressive (VAR) models using a common low-rank structure with sparse client-specific deviations.
- Develops a two-stage estimation procedure combining differentially private representation learning for shared components with local personalization.
- Establishes non-asymptotic error bounds to formally characterize the privacy-utility trade-off and provides proof of consistency for a ridge-type rank selection criterion.
