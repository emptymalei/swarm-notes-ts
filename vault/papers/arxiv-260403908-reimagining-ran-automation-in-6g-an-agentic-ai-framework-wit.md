# Reimagining RAN Automation in 6G: An Agentic AI Framework with Hierarchical Online Decision Transformer

**Authors**: Md Arafat Habib, Medhat Elsayed, Majid Bavand, Pedro Enrique Iturria Rivera, Yigit Ozcan, Melike Erol-Kantarci
**Date**: 2026-04-05
**Paper ID**: [arxiv:2604.03908](https://arxiv.org/abs/2604.03908)

## Summary

This paper presents an agentic AI framework for 6G radio access network (RAN) automation, centered on a Hierarchical Online Decision Transformer (H-ODT) that orchestrates various resource allocation, orchestration, and self-healing agents. The framework incorporates an Agentic RAG module to synthesize heterogeneous knowledge and utilizes generative AI-based time-series predictors for bi-level human intent validation. Evaluations demonstrate that the framework outperforms existing baselines in throughput, delay, and energy efficiency, while providing high-accuracy intent filtering and autonomous self-healing capabilities.

## Key Contributions

- Introduces an Agentic AI framework for 6G RAN automation orchestrated by a Hierarchical Online Decision Transformer (H-ODT).
- Implements a bi-level human intent validation methodology using generative time-series predictors, achieving 88.5% accuracy in filtering performance-degrading intents.
- Demonstrates robust self-healing capabilities, recovering 90% of system performance following performance-degrading events without human intervention.

## Key Concepts

- [[hierarchical-online-decision-transformer-h-odt]]: A hierarchical decision transformer architecture designed for real-time orchestration of heterogeneous AI agents in complex multi-agent systems.
