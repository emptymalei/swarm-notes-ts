# Large values in time series and additive combinatorics

**Authors**: Alex Iosevich, Vishal Gupta
**Date**: 2026-04-23
**Paper ID**: [arxiv:2604.21292](https://arxiv.org/abs/2604.21292)

## Summary

This paper provides a mathematical foundation for the observation that extreme values in real-world time series are often highly structured. By employing tools from additive combinatorics and discrete Fourier analysis, the authors demonstrate that when the Fourier ratio of a time series is low, its large values exhibit specific additive properties. The research validates this theoretical prediction through experiments on economic and meteorological datasets, showing that a small generating set is sufficient to reconstruct the structure of observed extreme values.

## Key Contributions

- Provides a rigorous theoretical framework using additive combinatorics and discrete Fourier analysis to explain the structured nature of extreme values in real-world time series.
- Proves that for time series with a small Fourier ratio, the set of largest values can be additively generated by a small set with coefficients in {-1, 0, 1}.
- Demonstrates through empirical validation on US inflation and Delhi climate datasets that a generating set of size 4--7 is sufficient to span large spectra of extreme values.
