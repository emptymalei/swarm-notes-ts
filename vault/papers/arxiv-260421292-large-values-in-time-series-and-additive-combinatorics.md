# Large values in time series and additive combinatorics

**Authors**: Alex Iosevich, Vishal Gupta
**Date**: 2026-04-23
**Paper ID**: [arxiv:2604.21292](https://arxiv.org/abs/2604.21292)

## Summary

This paper establishes a mathematical foundation for the observed structural patterns in extreme time series values using additive combinatorics and Fourier analysis. The authors show that when the Fourier ratio of a series is small, its largest values exhibit a specific additive structure, being generated by a small set of points. The theoretical predictions are validated through numerical experiments on real-world US inflation and climate datasets, confirming that minimal generating sets can explain dozens of extreme points.

## Key Contributions

- Formalizes the observation that extreme values in time series possess additive structure using discrete Fourier analysis and additive combinatorics.
- Proves that a small Fourier ratio implies that the set of large time series values can be generated by a small set of base points using {-1,0,1} coefficients.
- Demonstrates empirically on US inflation and Delhi climate data that large-value sets are spanning by generating sets of size 4--7.
