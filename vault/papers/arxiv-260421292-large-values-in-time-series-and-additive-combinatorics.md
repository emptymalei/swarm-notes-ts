# Large values in time series and additive combinatorics

**Authors**: Alex Iosevich, Vishal Gupta
**Date**: 2026-04-23
**Paper ID**: [arxiv:2604.21292](https://arxiv.org/abs/2604.21292)

## Summary

This paper provides a rigorous mathematical foundation for the observation that extreme values in real-world time series exhibit predictable, structured patterns. By leveraging tools from additive combinatorics and discrete Fourier analysis, the authors demonstrate that if the Fourier ratio of a time series is low, its largest values can be additively generated by a very small set. Empirical testing on inflation and climate data confirms that these extreme value sets are remarkably parsimonious, typically requiring only 4-7 generating elements.

## Key Contributions

- Establishes a mathematical framework using additive combinatorics and Fourier analysis to explain the structural patterns of large values in time series.
- Introduces the Fourier ratio as a tool to determine if large time series values are additively generated by a small set of coefficients.
- Validates the theory empirically showing that a small generating set of size 4-7 suffices to span dozens of extreme value points in US inflation and Delhi climate datasets.
