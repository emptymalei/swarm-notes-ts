# Machine Learning as a Transformative Tool for (Exo-)Planetary Science

**Authors**: J. Davoult, V. T. Bickel, C. Haslebacher, Y. Alibert, D. Angerhausen, C. Cantero, J. A. Egger, R. Eltschinger, Y. Eyholzer, E. O. Garvin, S. Gruchola, A. Leleu, S. Marques, Y. Zhao
**Date**: 2026-04-10
**Paper ID**: [arxiv:2604.09152](https://arxiv.org/abs/2604.09152)

## Summary

This review chapter examines the transformative role of machine learning in planetary and exoplanetary science, focusing on processing large, heterogeneous, and spatio-temporally inconsistent datasets. The authors categorize applications into three primary areas: sequence modelling for time-series data like light curves, pattern recognition for feature extraction and anomaly detection, and generative models for simulating planetary interior structures. The paper highlights the potential of these techniques to overcome current limitations in data interpretation, ultimately aiming to accelerate scientific discovery in the field.

## Key Contributions

- Reviews machine learning applications for planetary science, categorizing them into sequence modelling, pattern recognition, and generative modeling/emulation-based Bayesian analysis.
- Demonstrates how deep learning can address challenges in radial velocity time series, light curve analysis, and planetary interior structure prediction.
- Highlights the role of NCCR PlanetS in developing ML tools for heterogeneous, hyper-dimensional planetary datasets.
