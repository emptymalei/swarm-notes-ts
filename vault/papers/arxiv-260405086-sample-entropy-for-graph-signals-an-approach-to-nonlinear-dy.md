# Sample entropy for graph signals: An approach to nonlinear dynamic analysis of data on networks

**Authors**: Mei-San Maggie Lei, John Stewart Fabila Carrasco, Javier Escudero
**Date**: 2026-04-06
**Paper ID**: [arxiv:2604.05086](https://arxiv.org/abs/2604.05086)

## Summary

The paper introduces Sample Entropy for Graph Signals (SampEnG), a novel method to characterize nonlinear dynamics of signals defined on network structures. By generalizing classical sample entropy via topology-aware multi-hop embeddings, SampEnG provides a robust measure of conditional entropy for graph-structured data. Empirical validation shows the method successfully captures nonlinear features in synthetic signals and detects traffic flow phase transitions, offering a sophisticated alternative to existing Shannon-entropy-based graph signal metrics.

## Key Contributions

- Introduced SampEnG, a unified framework extending classical sample entropy to graph signals using topology-aware multi-hop embeddings.
- Demonstrated that SampEnG recovers known nonlinear dynamical features in synthetic path and grid signals.
- Validated that SampEnG on directed topologies effectively detects phase transitions in traffic flow, providing complementary information to Shannon-entropy metrics.
